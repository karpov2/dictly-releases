# Dictly — releases

**Dictly** — голос в текст для разработчиков (аналог Wispr Flow с фокусом на русскоязычных). Диктуете — получаете чистый, отформатированный текст прямо в активном приложении: в терминале, заметках, редакторе кода.

Это публичный репозиторий **сборок и авто-обновлений**. Исходный код приложения приватный.

## Скачать

👉 **[Последняя версия »](../../releases/latest)** — скачайте `.dmg` (macOS, Apple Silicon).

Установленное приложение **обновляется само**: видит новую версию и предлагает обновиться в один клик.

## Что в каждом релизе

- `Dictly_<версия>_aarch64.dmg` — установщик (macOS arm64)
- `Dictly_aarch64.app.tar.gz` + `.sig` — бандл и minisign-подпись для авто-апдейтера
- `latest.json` — манифест апдейтера

> Сборки пока без нотаризации Apple. Если при первом запуске macOS ругается, что приложение «повреждено»:
> `xattr -cr /Applications/Dictly.app`, затем открыть. Подпись Apple Developer ID — в планах.
